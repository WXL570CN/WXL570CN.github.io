React Query

🚀为TS/JS、React、Solid、Vue和Svelte提供强大的异步状态管理。

开始使用 →

声明性和自动

用手写你的数据获取逻辑已经结束了。告诉TanStack Query在哪里获取你的数据，以及你需要它有多新鲜，其余的都是自动的。它处理缓存，后台更新和陈旧的数据，开箱即用，零配置。

简单和熟悉

如果你知道如何使用承诺或async/await，那么你已经知道如何使用TanStack Query。没有全局状态需要管理，没有还原器、规范化系统或沉重的配置需要理解。只需传递一个能解决你的数据的函数（或抛出一个错误），剩下的就是历史了。

可扩展性

TanStack Query可以配置到每个观察者的查询实例，并有旋钮和选项来适应每个使用情况。它配备了专用的开发工具、无限加载的API和一流的突变工具，使更新数据变得轻而易举。不要担心，所有的东西都是为成功而预先配置的！
