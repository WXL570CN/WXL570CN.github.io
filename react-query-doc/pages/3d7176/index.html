useIsMutating

# useIsMutating

useIsMutating 是一个可选的钩子，它返回您的应用程序正在获取的突变数量（对于应用程序范围的加载指示器很有用）。

import { useIsMutating } from '@tanstack/react-query'// How many mutations are fetching?const isMutating = useIsMutating()// How many mutations matching the posts prefix are fetching?const isMutatingPosts = useIsMutating({ mutationKey: ['posts'] })

选项

  • filters?: MutationFilters: 变异过滤器
  • 上下文？：React.Context<QueryClient |未定义>
    • 使用它来使用自定义 React 查询上下文。否则，将使用 defaultContext。

Returns

  • isMutating：数字
    • 将是您的应用程序当前正在获取的突变数。
