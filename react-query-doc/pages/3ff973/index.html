QueryClientProvider

# QueryClientProvider

使用 QueryClientProvider 组件连接并向您的应用程序提供 QueryClient：

import { QueryClient, QueryClientProvider } from '@tanstack/react-query'
const queryClient = new QueryClient()
function App() {  return <QueryClientProvider client={queryClient}>...</QueryClientProvider>}

选项

  • 客户端：查询客户端
    • 必需的
    • 要提供的 QueryClient 实例
  • 上下文共享：布尔值
    • 弃用
    • 默认为假
    • 将此设置为 true 以启用上下文共享，这将跨窗口共享上下文的第一个实例和至少一个实例，以确保如果跨不同的包或微前端使用 React Query，它们都将使用相同的上下文实例，而不管模块范围。
  • 上下文？：React.Context<QueryClient |未定义>
    • 使用它来使用自定义 React 查询上下文。否则，将使用 defaultContext。
