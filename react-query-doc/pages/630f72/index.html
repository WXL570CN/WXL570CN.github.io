# InfiniteQueryObserver

[

# 无限查询观察者

](https://tanstack.com/query/latest/docs/react/reference/InfiniteQueryObserver#infinitequeryobserver)

InfiniteQueryObserver 可用于观察和在无限查询之间切换。

const observer = new InfiniteQueryObserver(queryClient, {  queryKey: ['posts'],  queryFn: fetchPosts,  getNextPageParam: (lastPage, allPages) => lastPage.nextCursor,  getPreviousPageParam: (firstPage, allPages) => firstPage.prevCursor,})
const unsubscribe = observer.subscribe(result => {  console.log(result)  unsubscribe()})

选项

InfiniteQueryObserver 的选项与 useInfiniteQuery 的选项完全相同。
